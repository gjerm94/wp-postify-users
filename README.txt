WP Postify Users is a plugin that inserts a regular WordPress post for every user that has registered on your site.

== Description ==

WordPress has tons and tons of plugins and themes that allow you to display your posts in a myriad of ways; however if you want to showcase a list of users your options are limited.
This plugin aims to solve that problem by making each user into a post object (Postify), allowing you to use these "User posts" in all the plugins and themes that does aim to display users.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-postify-users.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
